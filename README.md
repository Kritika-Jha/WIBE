# 🌟 WIBE – WellWork Intelligent Browser Extension

> **"Work smarter. Rest better."**

---

##  What is WIBE?

**WIBE** stands for **WellWork Intelligent Browser Extension** – but it’s not *just* a timer.

It's a **smart productivity partner** designed for people who spend long hours on screens. WIBE brings together:

- **Eye-care automation** (based on the 20-20-20 rule)
- **AI-powered assistance**
- **Distraction-blocking tools**
- **Posture correction alerts**
- **Screen-time analytics via connected website**

All packed into a single, elegant Chrome extension.

---

##  Why WIBE?

Most extensions solve **one** problem.  
WIBE solves a **whole system of problems** faced by digital professionals and students:

| ⚠️ Problem                              | ✅ WIBE Solution                                        |
|----------------------------------------|---------------------------------------------------------|
| Eye strain & burnout                   | Timed overlays with 20-20-20 vision protection built-in |
| Constant tab-switching for AI help     | AI assistant that opens directly in your current tab    |
| Time wasted on distractions            | Website blocker with one-click control                  |
| Posture fatigue (coming soon)          | Alerts to check and correct your posture                |
| Lack of productivity insight           | Connected site to track breaks & provide suggestions    |

## ✨ Features

### 👁️ 20-20-20 Eye-Care Timer
- Runs a **20-minute background timer**.
- Triggers a **20-second break overlay** with a visual timer.
- Includes **Start**, **Stop**, and **Skip** options.
- Loops automatically for continuous productivity cycles.
- Toggle to **enable/disable Eye Care mode**.

### 📌 Popup Dashboard
- Real-time display of timer.
- Manual Start/Stop controls.
- Toggle eye-care mode.
- Lightweight and responsive UI.

### 🔒 Website Blocker *(Coming Soon)*
- Block distracting websites with a simple URL entry.
- See and manage blocked sites.
- One-click unblock or delete.

### 🤖 AI Assistant *(Coming Soon)*
- Bring ChatGPT-style AI into any tab — no switching required.
- Enter your own API key for secure, custom AI use.
- Ask questions, get summaries, code help, or productivity advice on the fly.

### 🧍 Posture Alerts *(Planned Feature)*
- Integrate with posture-detection APIs via webcam.
- Prompt users to sit upright or stretch if posture declines.

### 🌐 Connected Web Dashboard *(Coming Soon)*
- Track how many breaks you’ve taken.
- Suggest exercises & reminders based on screen time.
- Download wellness reports or sync data across devices.

## 🧰 Tech Stack

- **React + TypeScript**
- **Vite** (fast frontend build tool)
- **Chrome Extensions Manifest V3**
- CSS Modules / Custom Styling

---
## 🧪 Load the Extension in Chrome

- Go to chrome://extensions
- Enable Developer Mode
- Click "Load Unpacked"
- Select the dist/ folder
---
## 🚀 Getting Started (Developers)

1️⃣ Clone the Repo
```bash
git clone https://github.com/your-username/wibe-extension.git
cd wibe-extension
```
2️⃣ Install Dependencies
```bash
npm install
```
3️⃣ Development Mode
```bash
bash
npm run dev
```
4️⃣ Production Build
```bash
npm run build
```
The production-ready build will be generated in the dist/ folder.
